Calculation workflow for Na3SbS4 hydration/recyclability

Overview
- Goal: Provide ready-to-run DFT input scaffolding and analysis utilities to quantify hydration/dehydration thermodynamics and interface reactions for Na3SbS4 and its hydrates (Na3SbS4·8H2O/9H2O), and prepare NEB templates for Na migration.

Contents
- 00_docs/
  - calculation_plan.md: Detailed plan and methods.
  - calculation_plan.pdf: PDF rendering of the plan (generated by utils/make_pdf.py).
- 10_structures/
  - Na3SbS4.cif (copied from model/)
  - Na3SbS4_yaosen.cif (Na3SbS4·8H2O, copied from model/)
- 20_vasp_inputs/
  - templates/INCAR.relax, INCAR.static, INCAR.NEB, KPOINTS.auto, POTCAR_MAP.txt
  - prepare_inputs.py: Generate POSCAR/KPOINTS/INCAR folders for relax/static from CIF.
  - cif_to_poscar.py: Minimal CIF parser -> VASP POSCAR writer (no external deps).
- 30_thermo/
  - reactions.yaml: Representative reactions for unhydrated and hydrated Na3SbS4 vs Na metal; hydration reactions.
  - compute_reaction_energies.py: Assemble reaction ΔE/ΔG from DFT totals and gas-phase μ(T,p).
  - energies.example.yaml: Example energy file schema to fill with your VASP results.
  - utils_thermo.py: Shomate-based μ for H2O and H2S; dew point ↔ RH helpers.
- 40_transport/
  - neb_template/
    - INCAR.NEB, KPOINTS, README.txt
  - make_neb.py: Skeleton to build NEB path folders given initial/final Na sites (expects you to specify indices).

Quick start
1) Copy POTCARs locally and set VASP_PSP_DIR to your PAW root (see 20_vasp_inputs/POTCAR_MAP.txt).
2) Convert CIF to POSCAR and prepare input sets:
   python3 20_vasp_inputs/prepare_inputs.py --cif 10_structures/Na3SbS4.cif --out 20_vasp_inputs/Na3SbS4
   python3 20_vasp_inputs/prepare_inputs.py --cif 10_structures/Na3SbS4_yaosen.cif --out 20_vasp_inputs/Na3SbS4_8H2O
3) Run relax → static; collect energies into 30_thermo/energies.yaml (use energies.example.yaml as a template).
4) Compute ΔE/ΔG vs RH/T:
   python3 30_thermo/compute_reaction_energies.py --energies 30_thermo/energies.yaml --reactions 30_thermo/reactions.yaml --T 298 --RH 0.68
5) Generate plan PDF anytime:
   python3 utils/make_pdf.py 00_docs/calculation_plan.md 00_docs/calculation_plan.pdf

Notes
- CIF→POSCAR uses a minimal parser designed for the included files. If you prefer pymatgen/ase, replace cif_to_poscar.py usage accordingly.
- POTCARs are not included; see mapping hints in POTCAR_MAP.txt.
- NEB path generation requires you to specify Na site indices; the script creates folder scaffolding and copies INCAR/KPOINTS.
